*easy_dbext_bundle.txt*

						*i_CTRL-CR*
<CTRL-CR>	Execute command (SQL, DML, DDL etc.) in |paragraph|. 
						*i_CTRL-C*
<F5>		Refresh (|:e| re-edit) the current file.
						*i_CTRL-F9*
<CTRL-F9>	Equalize all windows (|CTRL-W_=|).

						*i_F9*
<F9>		Execute DBResultsOpen (|dbext.vim|) if in a buffer other than Result; toggle
                its size (tall or short) when in it.
						*i_F10*
<F10>		Execute DBResultsClose (|dbext.vim|).

						*i_CTRL-kMinus*
<CTRL-kMinus>	|resize| the window with -5 as parameter.
						*i_CTRL-kPlus*
<CTRL-kPlus>	|resize| the window with +5 as parameter.
